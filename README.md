# Hrga

## Database Configuration

This project currently contains only a static frontend (`index.html`).

A local `.env` file has been added with Neon/Postgres connection settings for future backend integration.

If you add a server or API layer, use the environment variables from `.env` to connect to the database.

## Notes

- The team member count is already set to 8 people in `index.html`.
- The current project does not yet include server-side database synchronization logic.
- Use `.env.example` as a template if you want to create a separate environment file.
